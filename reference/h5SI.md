# Tools for handling h5Seurat indexes

Tools for handling h5Seurat indexes

## Usage

``` r
# S3 method for class 'h5SI'
DefaultAssay(object, ...)

# S3 method for class 'h5SI'
print(x, ...)
```

## Arguments

- x, object:

  An h5Seurat index (`h5SI`)

## Value

`DefaultAssay`: the assay set as the default assay

`print`: Invisibly returns `x`
