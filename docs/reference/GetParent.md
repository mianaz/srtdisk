# Get the parent of an HDF5 dataset or group

Get the parent of an HDF5 dataset or group

## Usage

``` r
GetParent(x)
```

## Arguments

- x:

  An HDF5 dataset or group

## Value

An [`H5File`](http://hhoeflin.github.io/hdf5r/reference/H5File-class.md)
or
[`H5Group`](http://hhoeflin.github.io/hdf5r/reference/H5Group-class.md)
object
